# OctoFishstick
A stock tracking application I build to further learn industry standard C, GTK and Data-Driven Design.

## Built With

| Component        | Tech Used           |
|------------------|---------------------|
| Language         | C23                 |
| GUI Toolkit      | GTK 4               |
| Networking       | libcurl             |
| JSON Handling    | json-c              |
| Build System     | CMake               |
| Compiler Tooling | Clang + Clangd      |
| Editor           | NeoVim              |
| Architecture     | Data-Driven Design  |
